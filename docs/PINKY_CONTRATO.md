## 1. OBJETO

Desarrollo del Sistema de Gestión de Nómina y Asistencia **PINKY** para control de asistencia, gestión de empleados, cálculo de nómina y reportes.

---

## 2. ALCANCE

### Incluye:

| Módulo | Funcionalidades Principales |
|--------|----------------------------|
| **Empleados** | Gestión completa, edición masiva, jefe superior obligatorio |
| **Asistencia** | Integración ZKTeco, puntualidad (desayuno), reglas de faltas/retardos |
| **Autorizaciones** | Horas extra, veladas, permisos (pre y post autorización) con evidencia |
| **Nómina Básica** | Puntualidad, faltas, retardos, vacaciones (visible para todos) |
| **Nómina Completa** | Horas extra, veladas, cenas, bonos (solo Admin/RH) |
| **Reportes** | Asistencia, prenómina, vacaciones, autorizaciones |
| **Logs** | Auditoría completa de acciones |

### Reglas Clave:

- Nómina semanal: Miércoles a Martes
- Retardos/Faltas: Cálculo mensual
- Salida anticipada = Falta (excepto con permiso)
- Permiso autorizado = Día completo trabajado
- Horas extra pendientes no visibles hasta autorizar
